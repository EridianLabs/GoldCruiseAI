üîß Key Parameters

Risk Management Parameters

Risk Per Trade: Percentage of account balance risked per trade. Lower values = lower risk, higher values = higher risk.

  • 0.5% = Conservative (beginners)
  • 1.0% = Moderate (recommended for most traders)
  • 2.0% = Aggressive (experienced traders)

Maximum Daily Drawdown: EA will stop trading if daily drawdown exceeds this limit. Helps protect your account from excessive losses.

Maximum Trades Per Day: Limits the number of trades the EA can open per day. Prevents overtrading and helps manage risk.

Trading Parameters

Confidence Thresholds: Minimum confidence score required to enter a trade. Higher values = fewer but higher quality trades.

Stop Loss & Take Profit Multipliers: Based on ATR (Average True Range). SL = 2x ATR, TP = 4x ATR by default. This ensures stops and targets adapt to market volatility.
