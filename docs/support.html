Frequently Asked Questions

What is the minimum account size?
We recommend a minimum account size of $500. However, you can start with smaller accounts using the FixedLot_0.01.set configuration. For best results, use at least $1,000 with the Moderate.set profile.
Which risk profile should I use?
  • Beginners / Small accounts ($500-$2,000): Conservative.set or FixedLot_0.01.set
  • Most traders ($1,000-$5,000): Moderate.set ‚≠ê (Recommended)
  • Experienced traders ($5,000+): Moderate.set or Aggressive.set
Always start with a demo account to test the configuration before going live.
Why isn't the EA trading?
Check these common issues:
  • AutoTrading button must be enabled (green)
  • Chart must be XAUUSD on M15 timeframe
  • Check that InpEnableTrading = true in EA inputs
  • Verify you have sufficient margin
  • Check if max daily drawdown limit reached
  • Review Journal tab for error messages
Can I use this EA on other symbols?
GoldCruiseAI is specifically optimized for XAUUSD (Gold) on M15 timeframe. While it may work on other symbols, we strongly recommend using it only on XAUUSD as it has been tested and optimized exclusively for gold trading.
What is the win rate?
Based on 25 years of backtesting (2000-2025), GoldCruiseAI achieved a 93.93% win rate with 626 total trades. However, past performance does not guarantee future results. Actual results may vary based on market conditions, broker, and account settings.
How do I load a set file?
  1. Download the desired .set file from the Set Files page
  2. Attach GoldCruiseAI to XAUUSD M15 chart
  3. Right-click EA ‚Üí Properties (or double-click EA)
  4. Click the "Load" button in the Inputs tab
  5. Select your downloaded .set file
  6. Click OK
Is there a monthly fee?
No! GoldCruiseAI is a one-time purchase. There are no monthly fees, no subscription costs, and no hidden charges. You pay once and own it forever, including all future updates.
Do I need an internet connection?
No, GoldCruiseAI works completely offline. It doesn't require any API keys, external services, or internet connection. Once installed, it runs independently on your MetaTrader 5 platform.
Can I modify the parameters?
Yes, you can modify all parameters to suit your trading style. However, we recommend:
  • Start with the provided set files
  • Test any changes on a demo account first
  • Use Strategy Tester to validate changes
  • Document your modifications
What if I'm not satisfied?
We offer a 1-month rental option ($99) so you can try GoldCruiseAI before committing to the full purchase ($599). If you're not satisfied with the rental, you're not obligated to purchase. For full purchases, please contact MQL5 support for their refund policy.
How often does it trade?
GoldCruiseAI focuses on quality over quantity. It typically makes 8-15 trades per day depending on your risk profile and market conditions. The EA only trades when multiple conditions align, ensuring high-probability setups. This is why it achieves such a high win rate.
Is it safe for beginners?
Yes! GoldCruiseAI is suitable for all experience levels. Beginners should use the Conservative.set or FixedLot_0.01.set profiles, which have very low risk per trade. Always start with a demo account to learn how the EA works before trading live.
