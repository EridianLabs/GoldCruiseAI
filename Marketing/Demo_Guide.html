Installation Guide

Step-by-Step Setup Instructions

Step 1: Download the EA

Purchase from MQL5 Marketplace

After purchasing GoldCruiseAI from the MQL5 Marketplace, the EA file will be automatically downloaded to your MT5 data folder.

Verify Installation

Open MetaTrader 5 and check the Navigator panel. You should see "GoldCruiseAI" under Expert Advisors.

  1. Open MT5 ‚Üí File ‚Üí Open Data Folder
  2. Navigate to: MQL5 ‚Üí Experts
  3. Copy GoldCruiseAI.ex5 to this folder
  4. Restart MT5

Step 2: Attach EA to Chart

Open XAUUSD Chart

In MetaTrader 5, open a chart for XAUUSD (Gold). Right-click on the chart and select "Symbols" ‚Üí search for "XAUUSD" ‚Üí double-click to add.

Set Timeframe to M15

Change the chart timeframe to M15 (15 minutes). You can do this by clicking the M15 button in the toolbar or pressing Alt+3.

Drag EA to Chart

From the Navigator panel, drag "GoldCruiseAI" onto your XAUUSD M15 chart. The EA input window will open automatically.

Configuration

Setting Up Your Risk Profile

Step 3: Load Risk Profile

Open EA Inputs

Right-click on the EA icon in the top-right corner of your chart ‚Üí select "Properties" (or double-click the EA icon).

Load Set File

In the Inputs tab, click the "Load" button. Navigate to your set files folder and select your preferred risk profile:

  • Conservative.set - 0.5% risk, fixed 0.01 lot
  • Moderate.set - 1.0% risk, adaptive sizing (recommended)
  • Aggressive.set - 2.0% risk, higher returns
  • FixedLot_0.01.set - Fixed lot size for small accounts
  • MaxProfit_2Percent.set - 2% risk variant
  • MaxProfit_3Percent.set - 3% risk variant

Verify Settings

Review all parameters to ensure they match your risk tolerance. Key settings to check:

  • Risk per trade percentage
  • Maximum daily drawdown limit
  • Maximum trades per day
  • Stop loss and take profit multipliers

Key Parameters

Understanding Important Settings

Risk Management Parameters

Risk Per Trade

Default: 1.0% (Moderate profile)

Percentage of account balance risked per trade. Lower values = lower risk, higher values = higher risk.

  • 0.5% = Conservative
  • 1.0% = Moderate (recommended)
  • 2.0% = Aggressive

Maximum Daily Drawdown

Default: 5.0%

EA will stop trading if daily drawdown exceeds this limit. Helps protect your account from excessive losses.

Maximum Trades Per Day

Default: 10

Limits the number of trades the EA can open per day. Prevents overtrading and helps manage risk.

Trading Parameters

Confidence Thresholds

Buy Confidence: 0.50 (default)

Sell Confidence: 0.30 (default)

Minimum confidence score required to enter a trade. Higher values = fewer but higher quality trades.

Stop Loss & Take Profit Multipliers

SL Multiplier: 2.0 (default)

TP Multiplier: 4.0 (default)

Multipliers based on ATR (Average True Range). SL = 2x ATR, TP = 4x ATR. This ensures stops and targets adapt to market volatility.

Session Filtering

The EA can be configured to trade only during specific sessions:

  • London Session: 08:00 - 16:00 GMT
  • New York Session: 13:00 - 21:00 GMT
  • Asian Session: Optional (usually avoided)

üí° Pro Tip

Session filtering helps avoid low-liquidity periods and can improve execution quality. The default settings optimize for London and New York sessions.

Q: Can I use this EA on other symbols?

A: GoldCruiseAI is optimized specifically for XAUUSD (Gold). While it may work on other symbols, optimal performance is achieved with XAUUSD M15.

Q: Do I need a VPS?

A: Yes, all parameters are adjustable. However, we recommend starting with the provided set files.

Q: What if the EA stops trading?
